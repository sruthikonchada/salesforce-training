# Day 8 - LWC Basics

## Overview

This project focuses on learning the fundamentals of Lightning Web Components (LWC) in Salesforce. A simple component named **bikeCard** was created to understand component creation, configuration, and deployment.

---

## What is LWC?

Lightning Web Components (LWC) is Salesforce's modern UI framework for building fast, reusable, and efficient web components using standard web technologies such as HTML, JavaScript, and CSS.

---

## Why Salesforce Uses LWC

- High Performance
- Reusable Components
- Modern Web Standards
- Better Developer Experience
- Easy Maintenance

## Component Breakdown

### bikeCard.html
Responsible for displaying the user interface.

### bikeCard.js
Contains the component's functionality and logic.

### bikeCard.js-meta.xml
Defines component configuration and visibility within Salesforce.

---

## Frontend vs Backend Logic

### Frontend
- Built using LWC
- Handles user interface and interactions
- Uses HTML, CSS, and JavaScript

### Backend
- Built using Apex and SOQL
- Handles business logic and database operations
- Processes and retrieves Salesforce data
